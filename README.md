# SoftGrowTech Music Player

A responsive web music player built with vanilla HTML, CSS, and JavaScript. Features a custom playlist, playback controls, volume adjustment, and a sidebar for easy navigation—optimized for both desktop and mobile.

## Features
- Custom playlist with album art
- Play, pause, next, previous controls
- Real-time progress bar and click-to-seek
- Volume slider with numeric indicator
- Sidebar playlist navigation (toggleable, mobile-friendly)
- Mobile responsive design with sidebar close button
- Keyboard shortcuts for playback and volume

## Tech Stack
- HTML5
- CSS3 (Flexbox, media queries)
- JavaScript (ES6)

## What I Learned
- DOM manipulation and event handling in vanilla JS
- Creating interactive UI components
- Responsive web design and mobile-first development
- Managing state and user experience across devices

## Getting Started
1. Clone the repository or download the files.
2. Add your `.mp3` files to `assets/music/` and album art images to `assets/covers/`.
3. Update the `songs` array in `js/script.js` with your music and cover info.
4. Open `index.html` in your browser.
